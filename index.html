Minecraft сервер на домашнем пк

Инструкция как установаить на свой компьютер игровой сервер Minecraft и играть с друзьями (2020-04-30)

Инструкции с ru-m.org

1) Скачать версию сервера скачать Forge https://ru-m.org/engine/download.php?id=12521
2) Распаковать архив в отдельную папку
3) Запустить файл start.bat - появится файл 'eula.txt', который нужно открыть и поменять занчение false на true и сохранить файл
4) Запустить файл start.bat и ждем завершения "Done"
5) Открыть файл "server.properies", найти параметр "online-mode=", поменять значение с true на false
6) Теперь, чтобы зайти на свой сервер, нужно скачать клиент с установленным Forge, например TLauncher. Скачать клиент, зайти в игру,в сетевую, добавить новый сервер IP - localhost

Инструкция как играть через Интернет с помощью Hamachi

Нужно поменять localhost на IP сервера, который добывается следующим образом
1) Скачать пробную версию Hamachi с офф.сайта
2) Следовать инструкциям по установки и регистрации. Когда программа установится - запустить ее, нажать на "Создать новую сеть", ввести любые данные, подключиться к сети. Создастся новая сеть
3) Устанавить Hamachi другим игрокам, зарегистрироваться, подключиться к созданной на предыдущем этапе сети - ввести название и пароль. В Hamachi администратора отобразится ник нового подключенного игрока
4) Скопировать адрес IPv4 из Hamachi администратора сервера. Открыть файл в папке Minecraft сервера "server.properties". Найти параметр "server-ip=", вставить скопированный адрес IPv4 из Hamachi
5) Запустить сервер Minecraft - start.bat
6) Скопированный адрес IPv4 из Hamachi отправить игрокам, чтобы они ввели его как адрес сервера и подключились к игре!
